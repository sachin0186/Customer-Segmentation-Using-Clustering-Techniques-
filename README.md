# Customer-Segmentation-Using-Clustering-Techniques-
This project demonstrates mall customer segmentation using machine learning. It applies K-Means, DBSCAN, and Affinity Propagation for clustering based on age, income, and spending. Includes EDA, hyperparameter tuning, and visualizations using Scikit-learn, Pandas, Seaborn, and Plotly. Results: 5-6 actionable clusters.
